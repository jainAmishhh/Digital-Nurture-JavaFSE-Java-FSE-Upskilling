Bootstrap Structure and Files

CSS Folder

Contains Bootstrap stylesheets such as bootstrap.min.css used for layout, colors, spacing, typography, and components.

JS Folder

Contains JavaScript files like bootstrap.bundle.min.js used for modals, accordions, tooltips, dropdowns, and other interactive components.

Icons Folder

Contains Bootstrap Icons used for adding graphical symbols and improving UI appearance.

This accordion works because bootstrap.bundle.min.js is loaded.
